Dora / Doraemon Agent System is the coordination layer for Weiyu's personal AI lab.

It helps translate intent into concrete plans, route work to specialized MiniDora agents, collect results, and bring decisions back to Weiyu when judgment is required.

The current public website uses an original AI companion visual language. If Weiyu later chooses a specific Doraemon image for personal use, the site can swap that into the Dora entrance as an intentional asset decision.

## Boundary

The public Dora page is an informational guide. It cannot read private memory, trading data, credentials, local files, email, calendar, or internal agent runtimes.
